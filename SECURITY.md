# Política de Segurança

Se você encontrar uma vulnerabilidade, por favor reporte de forma responsável.

## Como reportar

- Envie um e-mail para: security@braindeveloper.local
- Inclua detalhes claros para reprodução.

## Suporte de versões

Este projeto é estático. Correções de segurança podem ser feitas a qualquer momento na branch principal.
