# Código de Conduta

Este projeto adota o Contributor Covenant, versão 2.1.

## Nosso Compromisso

Prometemos criar uma comunidade aberta e acolhedora para todos.

## Nossos Padrões

Comportamentos que contribuem para um ambiente positivo incluem:

- Respeito e empatia.
- Feedback construtivo.
- Colaboração.

Comportamentos inaceitáveis incluem:

- Assédio, discriminação ou discurso de ódio.
- Ataques pessoais.
- Divulgação de informações privadas.

## Aplicação

Casos de abuso podem ser reportados para: security@braindeveloper.local

## Atribuição

Este Código de Conduta é adaptado do Contributor Covenant 2.1:
https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
